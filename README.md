# 📖 Deskripsi

**Reminder Tugas Mahasiswa** adalah aplikasi Android yang dirancang untuk membantu mahasiswa mengelola tugas kuliah beserta deadline-nya.

Aplikasi ini memungkinkan pengguna menambahkan, mengubah, menghapus, serta menerima pengingat otomatis sebelum batas waktu tugas berakhir.

# 🚀 Instalasi

Clone repository

```bash
git clone https://github.com/ibnusab/ReminderTugasMahasiswa.git
```

Masuk ke folder

```bash
cd ReminderTugasMahasiswa
```

Buka menggunakan

```
Android Studio
```

Sync Gradle

```
Sync Project with Gradle Files
```

Jalankan aplikasi pada Emulator atau perangkat Android.
